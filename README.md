
# 电商项目技术栈

前端技术栈：Vue 3 + Element UI + Pinia
后端技术栈：Springboot + Mybatis + Mysql + Redis（预计将使用）

# 访问地址
https://www.linzhou.tech

## 功能模块

- 商品购买  
- 商品评价、点赞  
- 购物车模块  
- 支付宝支付沙箱  
- 我的订单  
- 我的板块  
- 登录注册
